3. 03_finishing_position_regression.ipynb

focus on predicting finishing position instead of binary points.

Best techniques:

multiple linear regression as a baseline
regularized regression
tree-based regression
ordinal framing discussion

Important note: finishing position is ordered and bounded, so plain linear regression is useful as a baseline, but not perfect. You can still use it to demonstrate regression diagnostics and residual analysis very effectively.
